gl-oo-interface

OpenGL Objected-Oriented Interface that provides classes and functionalities to OpenGL 4.0 with custom shaders and GLUT.

Welcome to gl-oo-interface!

This multi-platform library provides classes for creating and rendering meshes using the shader-based OpenGL 4.0. It is currently under development. I started implementing to help with my projects during Computer Graphics course at USC. Check out two applications here.

It includes:

  • Mesh: geometric data which is uploaded to GPU;
  • Texture: images loaded from .jpeg files that are settable to shaders;
  • Material: defines colorful materials according to Phong shading model;
  • SceneObject: encapsulates mesh, texture and material for rendering according to its position, orientation and lighting;
  • Camera: implemented to use as first-person view camera, editor-camera or objected-reference camera;
  • Light: created accordingly to Phong shading model;
  • Scene: makes possible displaying multiple objects, lights, camera and changing current cameras.
  • Basic Object Library: has many primitive objects such as rgb-axis, grid, textured spheres, textured terrain (optionally with heightmap loader) to render with small number of calls.
  • Video Recorder: records and takes screenshots during program execution;
  • GlutProgram: an abstract class which has an interface for general GLUT programs (keyboard, mouse).
